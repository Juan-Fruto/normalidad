Prctica 06 Prueba estadstica de normalidad en un conjunto de datos con el lenguaje R

Muchas de las pruebas de hiptesis que se hacen en el rea de ingeniera asumen que los datos de la variable dependiente tienen una distribucin normal, es decir, que la forma en la que sus valores se distribuyen en el plano se ajusta a una campana de Gauss. Por esta razn, una labor importante del ingeniero es determinar si la distribucin de una variable experimental es normal o no. Hay dos mtodos generalmente aceptados para analizar normalidad, uno es grfico y otro es usando una prueba de normalidad, de las cuales existen varias. Para ilustrar ambos tipos de anlisis trabajaremos con dos data sets en esta prctica.

El primer data set a trabajar almacena informacin del INEGI, la cual contiene el porcentaje de poblacin econmicamente activa, por cuatrimestre, en Mxico desde 2006 hasta 2019. Esa tabla ya la has trabajado en prcticas previas. Antes de abrir el data set, no olvides instalar el paquete para lectura de archivos .xlsx e invocar la librera para su lectura. Usa: 

library("readxl")

Ahora abre la tabla de datos del INEGI para crear el primer data set en R que analizaremos en la prctica (no olvides cambiar la ruta de acceso a tu archivo): 

df_INEGI <- read_excel("C:/Users/juan_/OneDrive - Universidad de Colima/Escritorio/6to/practica 6 normalidad - MdD/datasets/PobOcupada2019Mx_INEGI.xlsx")

Ahora analizaremos la posible normalidad de esos datos utilizando el mtodo grfico. En este caso, la idea es graficar la distribucin de frecuencias de los datos de la columna Porcentaje del libro de hoja de clculo datosINEGI.xlsx y dibujar una grfica de densidad de frecuencias para ver la forma en que se visualiza la distribucin, primero debes crear el grfico de frecuencias con la funcin hist() y la curva de densidad de frecuencias con la funicn lines : 

hist(df_INEGI$Porcentaje, probability=T, main="INEGI Frequency histogram", xlab="Economically active population")
lines(density(df_INEGI$Porcentaje), col='orange')

Con lo anterior, deberas visualizar algo similar a lo que se muestra en la figura 1. Como puedes ver, la distribucin de los datos de la variable porcentaje no se apega a una distribucin normal.


Figura 1. Histograma de frecuencias del data set de INEGI.


Ahora analizaremos la distribucin utilizando una prueba de normalidad llamada Shapiro-Wilks, la cual se basa en la hiptesis nula de que una muestra proviene de una distribucin normal. Si no se programa otra cosa, en R el nivel de significancia de esa prueba se fija 0.05. Para realizar entonces la prueba de hiptesis utilizaremos la funcin shapiro.test() : 

shapiro.test(df_INEGI$Porcentaje)

    Shapiro-Wilk normality test

data:  df_INEGI$Porcentaje
W = 0.94888, p-value = 0.01381

Para determinar si el resultado de la prueba Shapiro-Wilk indica la normalidad de la distribucin debemos analizar si el valor p obtenido (p-value) supera el nivel de significancia fijado, el cual como indicamos anteriormente es 0.05. Como observas en el resultado del comando, p-value = 0.01381 es menor que 0.05, por lo que se rechaza (o no se acepta) la hiptesis nula y se acepta la hiptesis alterna, determinando que la distribucin no es normal. El resultado de este mtodo coincide con lo que observas en la figura 1.

Ahora veamos otro caso. En esta parte de la prctica importaremos una hoja de clculo con datos de temperatura de un sensor de internet de las cosas que registra temperaturas al interior de un refrigerador. Primero debemos abrir el libro con los datos de temperaturas, que se encuentra en el archivo Temp_data.xlsx para crear un data set en R llamado datosTemp :

Siguiendo con la metodologa indicada anteriormente, para analizar la normalidad de un dato, en este caso la columna del dataset llamada gradosC , graficaremos las frecuencias y su curva de densidad para ver el comportamiento. As, teclea los siguientes dos comandos: 

hist(df_temperature$gradosC, probability=T, main="Fridge temprerature frequency histogram", xlab="celcius degrees")
lines(density(df_temperature$gradosC), col='brown')

El resultado de los comandos anteriores se ilustra en la figura 2, en la que puede observarse que la distribucin de los datos de temperatura se acercan a una distribucin normal.


Figura 2. Histograma de frecuencias del data set de temperaturas.

Ahora procederemos a hacer el anlisis con la prueba de normalidad: 

shapiro.test(df_temperature$gradosC)

    Shapiro-Wilk normality test

data:  df_temperature$gradosC
W = 0.98994, p-value = 0.6599

El comando anterior debe mostrarte lo siguiente: Shapiro-Wilk normality test data: datosTemp$gradosC W = 0.98994, p-value = 0.6599

Como observas en el resultado, el p-value = 0.6599 es mayor que el nivel de significancia de 0.05, por lo que se acepta la hiptesis nula de que la distribucin es normal.
